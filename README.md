# WeeklyReport
週報アプリ
https://weekreport5430.web.app/

## 使い方
- 週報の内容の週の月曜日を選択し、反映を押す
- 実施時間と内容等々を入力する(10時ちょうどなら、分は0または空白でおけ)
- 休みの場合は、時間は空白でおけ
- 研究時間はgmailリンクから飛び、先週の週報メールの総研究時間を確認し、研究時間に入力する(要改善)
- 作成を押すと週報のタイトルと内容が表示される
